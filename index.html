¿Cómo implementamos Firebase a nuestro proyecto de Android?

Hay dos formas:

  • Opción 1: (recomendado) Usar la consola de Firebase para configurar todo el flujo del trabajo.
  • Opción 2: Usar el asistente de Firebase para Android Studio (se necesita configuración adicional).

Opción 1:

Paso 1:Crear un proyecto de Firebase.

Antes de poder implementar Firebase a tu aplicación, necesitas crear un proyecto de Firebase. Para ello se necesita acceder a la consola de Firebase. Necesitarás una cuenta de Google para poder acceder a ella. Una vez allí, haz click en "Añadir un proyecto", luego pon tu nombre del proyecto.

Paso 2:Registrar tu aplicación.

Una vez creado el proyecto en Firebase puedes añadirla a tu proyecto en Android Studio.

1. Click en el icono de Android para correr la configuración del flujo de trabajo.

2. Ingresa el id de la aplicación en el campo delnombre de paquete de Android.

3. (opcional) Ingresa información adicional al flujo de trabajo.

4. Click Registrar aplicación.

Paso 3:Añade un archivo de configuración de Firebase.

1. Configurar Firebase en la app:

a. Descargar google-service.json para obtener el archivo de configuración de Firebase para Android.

b. Mueve el archivo de configuración en el modulo directorio de la app.

2. Para habilitar los productos de Firebase a la aplicación, añade el google-service.plugin al Gradle.

Paso 4:Añadir Firebase SDK a la app.

1. En el Gradle, añadir la dependencia de Firebase SDK.

2. Añadir las dependencias de las otras librerías de Firebase.

3. Sincronizar la app asegurando que Android Studio esté actualizado.

4. Correr la app.

Opción 2:

1. Abrir Android Studio.

2. Elige Herramientas > Firebase para abrir la ventana del asistente.

3. Agregar uno de los productos de Firebase.

4. Click "Conectar con Firebase" para registrar la app.

5. Verificar que todos los servicios estén actualizados.

6. Sincronizar la app para verificar que todo tenga la versión necesaria.

7. Configurar Analythics para compartir los datos de la app.

8. Correr la app.
